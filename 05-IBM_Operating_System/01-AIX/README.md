# AIX

## 简介
IBM 为 IBM Power 体系架构开发的一种基于开放标准的 UNIX 操作系统，具备安全性、可扩展性和稳健性。

AIX产品文档：[https://www.ibm.com/support/knowledgecenter/ssw_aix](https://www.ibm.com/support/knowledgecenter/ssw_aix)
