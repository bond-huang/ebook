# InfoSphere_Data_Replication

## 简介
&#8195;&#8195;IBM Data Replication-CDC Replication是一种复制解决方案，可在数据库更改发生时捕获它们并将它们交付到目标数据库、消息队列或ETL解决方案，例如基于IBM Data Replication管理控制台中配置的表映射的IBM DataStage®图形用户界面应用程序。
