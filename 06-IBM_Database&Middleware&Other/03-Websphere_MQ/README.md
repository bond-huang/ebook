# Websphere_MQ

## 简介

&#8195;&#8195;IBM MQ 是一款企业级消息传递解决方案，能够安全、可靠地连接多个平台的应用、系统、服务及文件。MQ 能够通过队列系统，在这些作业点之间传输数据，确保在网络或应用出现故障时，实现稳定交付。
