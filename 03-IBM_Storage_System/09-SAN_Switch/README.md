# SAN_Switch

## 简介

SAN全称Storage Area Networks，SAN Switch通俗点讲就是连接接存储和主机的光纤交换机。

IBM SAN Switch有两个系列：
- SAN b 系列：OEM博科
- SAN c 系列：OEM思科

官方网站：[https://www.ibm.com/cn-zh/it-infrastructure/storage/san](https://www.ibm.com/cn-zh/it-infrastructure/storage/san)
