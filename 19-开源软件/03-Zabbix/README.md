# Zabbix
&#8195;&#8195;Zabbix由Alexei Vladishev创建，目前由Zabbix SIA主导开发和支持。Zabbix是一个企业级的开源分布式监控解决方案。Zabbix是一款监控网络的众多参数以及服务器、虚拟机、应用程序、服务、数据库、网站、云等的健康和完整性的软件。Zabbix使用灵活的通知机制，允许用户为几乎任何事件配置基于电子邮件的告警，以实现对服务器问题做出快速反应。
