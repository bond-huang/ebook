# Bootstrap

## 简介
&#8195;&#8195;Bootstrap来自Twitter，是目前最受欢迎的前端框架。Bootstrap是基于HTML、CSS、JAVASCRIPT，用于开发响应式布局、移动设备优先的WEB项目。

官方网站：[https://getbootstrap.com/](https://getbootstrap.com/)
