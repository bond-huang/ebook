# Shell脚本

## 简介

&#8195;&#8195;Shell类型有多种，主流Linux系统一般是Bourne Again Shell (简称bash)，AIX系统中是Korn Shell（简称ksh）。不同Shell类型中Shell脚本有所差别，不过AIX安装个bash就可以切换到bash。
